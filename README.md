# Draylix
加密代理，客户端用go语言fyne框架，服务端采用netty

- 支持Windows
- 支持https，http，socks5 混合代理
- 支持一键开启系统代理
- 采用aes加密，支持客户端验证
- 支持消息时间戳检查
- 可显示实时流量，总共流量，服务器延迟
